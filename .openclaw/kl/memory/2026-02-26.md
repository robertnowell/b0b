## Key Learnings

### filter_reviews tool is unreliable
- Works sometimes (Gary Westby removal worked!) but fails consistently for Ryan Woodhouse, Ryan Moses, and any "top N by score" filtering
- **Best practice**: After pipeline, check if staff review still exists with `grep`, then use `edit` tool for surgical HTML removal

### Hero intro pattern
- When buyer review = hero intro, pass it via `Intro:` field in inputs and add custom instruction to remove from wine block
- Pipeline's `hasHeroSection` rubric will pass if intro is in the `Intro:` field (not as custom instruction replacing the hook)
