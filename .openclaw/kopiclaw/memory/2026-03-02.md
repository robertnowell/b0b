
## Infinite PR Feedback Loop (Hotfix)
- Fix agents post "Feedback Addressed" comments mentioning @kopi-claw on PRs
- gh-comment-dispatch picked those up as new feedback → dispatched another fix agent → infinite loop
- **Hotfix:** Added BOT_AUTHORS filter in gh-poll-process.py to skip comments from kopi-claw and github-actions[bot]
- Pushed directly to main (a3397911a) — emergency fix to stop active loop
- **Lesson:** Any bot that processes its own output creates feedback loops. Always filter self-authored content.
